Les didacticiels

Les didacticiels sont un excellent moyen de démarrer. Définissez le filtre Type pour démarrer rapidement les didacticiels qui vous présentent les bases de la création d’esquisse, de la modélisation de pièces, de la création d’ensembles et de la documentation de vos conceptions dans un dessin.

Vous pouvez accéder à ces didacticiels en cliquant sur la galerie de didacticiels sous l’onglet Mise en route, groupe de fonctions Ma page d’accueil sur le ruban. Remarque : Les didacticiels ne sont pas disponibles dans Inventor LT.

Une fois que vous êtes familiarisé avec les fonctions de base, définissez le filtre Lieu sur Tous disponibles et le filtre Type sur Tous afin d’afficher les didacticiels qui couvrent d’autres sujets, comme la conception de tôlerie. Remarque : Vous devez disposez d’une connexion Internet active pour afficher Tous disponibles.

Vous pouvez modifier l’affichage de la galerie en cliquant sur l’une des options suivantes :

  • Tous disponibles : affiche tous les didacticiels que vous pouvez télécharger.
    • La galerie est mise à jour lorsqu’un nouveau didacticiel est partagé publiquement avec la communauté.
    • Modifiez le filtre des langues pour afficher les didacticiels disponibles dans d’autres langues.
    • Vous pouvez sélectionner plusieurs langues.
    • Les didacticiels que vous avez téléchargés précédemment sont également affichés dans cette vue. Une bannière signale qu’ils ont été installés.
  • Installés : affiche uniquement les didacticiels installés en local.
  • Mes didacticiels : affiche les didacticiels que vous créez.
  • Partagé avec moi : affiche les didacticiels partagés de façon privée avec vous.
